# Автоматизация развертывания и мониторинга веб-форума для обсуждения компьютерных игр

## Описание

Данный проект предназначен для автоматизации процессов развертывания и мониторинга веб-форума.

## Используемые технологии

- Docker
- Docker Compose
- GitHub Actions
- Nginx
- Prometheus
- Grafana
- MariaDB

## Архитектура системы

Пользователь → Nginx → Веб-форум → База данных

GitHub → GitHub Actions → SSH → Docker Server

## Реализованные возможности

- контейнеризация приложения;
- автоматическое развертывание;
- настройка CI/CD;
- мониторинг состояния системы.
